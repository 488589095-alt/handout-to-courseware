## 分页规则（按题型）
阅读理解 / 七选五 / 语法填空 / 主观题翻译 = **每页 1 题**；完形填空 = 每页 3–4 题（一段覆盖的题量，≤4）。
每题一律 "题页 → 答页" 两页揭示。

## 顺序规则
1. 揭示两页式：所有讲解/练习先题(留白)后答。
2. 分析块前置步骤分隔页并高亮当前步（continuation 的三步）。
3. 参考范文放所在 PART 最后。
4. Leading-in（continuation）放 PART1 分隔之后。
5. 节(sections)：封面·目录 / PART1 / PART2 / (PART3) / 结束 —— 可选加 PowerPoint 节。
