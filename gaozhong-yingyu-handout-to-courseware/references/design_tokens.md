# 设计 Token（从 原始课件模板 + 标杆 一次性提炼；渲染所有讲次共用）

## 画布
- 16.67 × 12.5 in（模版页面尺寸）；正文留边 ≈ 0.9–1.0 in；页底注释区 y≈11.5

## 字体
| 用途 | 字体 |
|---|---|
| 标题/标签/目录 | 阿里巴巴普惠体 B |
| 中文正文/展示/解析 | 可口可乐在乎体 楷体 |
| 英文正文/题目/选项 | Arial |

## 颜色
| Token | 值 | 用途 |
|---|---|---|
| PURPLE | #7A2E8E | 页标题、强调、目录圆点 |
| PURPLE2 | #A076CE | 浅紫描边/箭头/AI注释 |
| INK | #1A1A1A | 正文 |
| RED | #FF5050 | 【答案】 |
| GOLD | #F2A900 | 难度★ |
| CARD_BG/CARD_LN | #FFF7EC / #E6C9A8 | 暖色内容卡片(原文概括/细节) |
| HL_BG/HL_LN | #E8F5E9 / #46A35E | 正确选项高亮 |

## 字号基准
页大标题36-48 · 大字节名/题源60-72 · 翻译中文30-34/英文26-27 · 题目选项26 · 解析20 · 语篇18(en) · 对照16/17 · AI注释13-14

## 模版 fixture（原始课件模板.pptx, 1-based）
封面=2（Text 0=讲次/Text 1=标题）· 空白内容页=4（清空shape留品牌背景）· PART分隔=5/7/9（Text 1=PART N/Text 0=小标题）· 结束=12 ·（课前测=1/课后测=11 壳，内容需外部输入）

## 版式模式
- 揭示两页式 = 第一页(题/留白) + 第二页(同版式+答案/填充)
- 步骤分隔页 = 3 卡片，当前步骤紫底白字，其余浅紫
- 图片只允许：讲义自带图(Knowledge Map) + 模版自带装饰；其余一律不插
