# 字号 + 字符兼容性规则

## 字号硬约束（小学课件甲方要求）

```
正文文本框：微软雅黑 24（硬要求，不要轻易改）
表格      ：按列数自适应
  - ≤3 列：18
  - 4-5 列：14
  - ≥6 列：12
占位符标题：沿用模板默认（template 已经设好字号，不要覆盖）
封面大标题：沿用模板默认（通常 36-44）
结束页大字：54（"下节课再见！"等）
```

## 为什么正文必须 24

小学生（三/四/五年级）课件，正文字号小于 24 在投影上看不清。这是高途运营 + 教研多次踩坑后定下的硬指标。

实现方式：

```json
// slide_structure.json 里所有 add_text_boxes
{
  "text": "...",
  "left": 1.4, "top": 1.0, "width": 10.5, "height": 5.7,
  "font_size": 24    // ← 一律 24
}
```

## 字号 24 下的内容容量

```
文本框尺寸                   ≈ 字符容量（24 号汉字）
─────────────────────────────────────────────
width=10.5 height=5.7        200-250 汉字
width=10.5 height=3.0        100-130 汉字（上半，配下半表格）
width=11.0 height=4.5        180-220 汉字（例题解答区）
```

内容超出时按优先级处理：
1. **拆 P**（教学设计若给 P 范围，本意往往就是分页）
2. **删冗余**（去掉"师：xxx 生：xxx"重复对话，保留观点）
3. **缩字号**（22 → 20，要先和老师确认，因为违反硬约束）

## 字符兼容性（v1 重大踩坑）

### Unicode 上标字符是杀手

```
禁用                              改用
─────────────────────────────
² ³ ⁴ ⁵ ⁶ ⁷ ⁸ ⁹  ⁰ (U+207x)        ^2 ^3 ^4...
ⁿ                                  ^n
```

**为什么禁用**：微软雅黑（以及大部分 CJK 字体）没有 U+207F 等上标字形。WPS/Office 渲染时会 fallback 到其他 CJK 字符集（韩文、中日韩兼容汉字），结果显示成乱码。

实际事故（v1.4 → v1.5）：

```
✗ 表格里写 "2ⁿ层"  → 显示成 "2쒊层"（韩文 잘 在 PPT 里乱出来）
✓ 表格里写 "2^n 层" → 正常显示
```

### 已验证安全的字符（可放心用）

```
✅ 安全（微软雅黑都有字形）：
  数学：°, ×, ÷, →, ←, ↑, ↓, ≤, ≥, ≠, ∞, ±
  序号：①②③④⑤⑥⑦⑧⑨⑩
  分隔：─, ━, │, ┃
  emoji：🎯🐮🪜🐰🐜🍎♟️♦️📚📝💡✨📐📋🔍🎮 等

⚠️ 谨慎使用（部分模板字体不全）：
  特殊上下标 U+207x / U+208x
  古希腊字符（小学课件用不到，建议规避）
  生僻数学符号（∑、∫、∂ 等，画大型公式用 OMML，不用 Unicode）
```

### 数学公式渲染

字号 24 下，简单公式直接用 ASCII：

```
✓ 段数 = 2^n + 1
✓ 折痕数 = 2^n - 1
✓ f(n) = f(n-1) + f(n-2)
✓ (200-150) ÷ (25-15) = 5 份/天
```

复杂公式（多行、矩阵、求和）才走 OMML（继承 `ppt-new-template` 的字体规则，详见该 skill 的 references/script_architecture.md）。

## 字体设置必须显式

`render_pptx.py` 的 `_set_run_lang` 会给每个 run 设置：

```xml
<a:rPr lang="zh-CN">
  <a:ea typeface="微软雅黑"/>
</a:rPr>
```

不显式设的话，会用模板默认（可能是黑体或宋体），效果不一致。

## 自适应字号示例（python 代码）

把所有表格字号根据列数自动调整：

```python
def auto_table_font(data):
    cols = max(len(r) for r in data) if data else 0
    if cols <= 3:  return 18
    if cols <= 5:  return 14
    return 12
```
