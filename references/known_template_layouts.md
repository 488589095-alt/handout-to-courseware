# 已注册的 PPT 模板版式

## 三年级A+ 春季 课件模板

**路径**：`/Users/gaotu/Documents/git/product-design-space/三年级A+ 春季 课件模板 .pptx`

**适用范围**：小学三/四/五年级 数学（A+ 春季/秋季/暑期/寒假 通用）

### 版式清单（按主用途分组）

#### 封面 / 目录 / 标题类

| 版式名 | 用途 | 占位符 |
|---|---|---|
| `封面` | 第 1 张：讲义封面 | ph[12]=标题(主题名)，ph[13]=讲次("第N讲")，ph[14]=季节("春季")，ph[19]=老师("主讲老师：XXX") |
| `目录` | 第 2 张：本学期讲义目录 | ph[11]-[16]：各讲标题列表 |
| `标题-文字少` | 章节大标题页 | ph[16]=标题文字 |
| `标题-文字多` | 章节多文字（如知识阶梯）| ph[16]=多行文字 |

#### 内文类

| 版式名 | 用途 | 占位符 |
|---|---|---|
| `内文1` | 例题页（题干+解析）| ph[11]=题号("例 N")，ph[16]=题干和解析 |
| `1_内文1` | 例题页（变体）| ph[11]=题号，ph[15]=解析图，ph[16]=题干 |
| `2_内文1` | 例题页（变体）| 同上 |
| `内文2` | 故事页 / 思考题 / 探究 | ph[11]=主文本，ph[12]=辅文本（答案/提示）|
| `内文3` / `1_内文3` | 练习题页 | ph[11]=题号("练 N")，ph[10]=题干，ph[13]=tag |
| `内文4` | 综合页（如本讲总结）| ph[10]-[14] 多占位符 |
| `内文5` | 单内容大页 | ph[10]=主内容 |
| `内文6` | 多内容布局 | ph[10]-[13] |

#### 空白 / 总结类

| 版式名 | 用途 | 占位符 |
|---|---|---|
| `空白页1` / `1_空白页1` | 笔记页 / 方法总结 | ph[11]=标题（如"笔记"），ph[12]=正文 |
| `空白页5` | 模块封面（如 "01. 模块名"）| ph[10]=序号("01.")，ph[11]=模块名 |
| `思维导图` / `1_思维导图` | 知识图谱 | ph[11]=标题，ph[12]=图谱内容 |

#### 结束类

| 版式名 | 用途 |
|---|---|
| `结束页` / `1_结束页` | 最后一页：下节课再见 |

### 占位符填写规则

1. **中文字体必须**：所有 run 都要 `set ea typeface="微软雅黑"`
2. **公式必须 OMML**：用 LaTeX 转 OMML，禁止纯文本 `F = ma`
3. **占位符缺失时**：从 layout 复制 sp 元素补齐（见 ppt-new-template 的实现）
4. **未使用版式不删除**：模板原有的 23 张样例 slide 删掉，但 layout 保留供 add_slide 调用
